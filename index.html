  3. What are the main services in the Internet?

    1. World Wide Web (WWW)

      2. This is a large collection of electronic documents connected to the Internet and stored on computers around the world.

      A user who uses www can access web sites through a Web Browser.

    2. File Transfer Protocol (FTP)

      3. FTP is used to transfer large files of one computer in the Internet to another computer.

    3. Remote Access

      4. Controlling and operating a computer in the Internet remotely is performed through remote access. Following are certain functions which can be performed through Remote Access after accessing the Internet.
      1. Installing software of server computers to client computers.
      2. Fixing problems on client computers

    4. File Sharing

      5. It is because of the file transfer service that thousands of users can connect to a website simultaneously.

    5. Streaming of Media

      6. Basic Streaming process

      1. A user visits a Web page hosted on a Web server and finds a file he'd like to see or hear.
      2. The Web server sends a message to streaming media Server, requesting the specific file.
      3. The streaming server streams the file to the user's computer, bypassing the Web server.
      4. The client software on the user's computer decodes and plays the file.

  4. How the Internet is used in daily life?

    5. The internet is used in daily life in many ways.

    • Communication - Sending and receiving emails, instant messages and making video calls.
    • Entertainment - Watching movies, listening to music, playing games and browsing social media.
    • Education - Accessing online courses, research papers and educational resources.
    • Shopping - Purchasing goods and services online.
    • Banking and Payments - Managing bank accounts, transferring money and paying bills online.
    • Healthcare - Booking doctor appointments and accessing medical information online.
    • Government Services - Applying for official documents,paying taxes and accessing public services online.

  5. Role of Web Browsers.

    6. A web browser is software that allows users to access, view, and interact with websites on the internet safely and efficiently.

    • Access websites -Opens websites using their web addresses (URL).
    • View web pages - Shows text,images ,videos and other content.
    • Interact with web content - Allows users to interact with web elements like links, forms, and buttons.
    • Run web applications - Supports interactive websites such as email, online banking, and social media.
    • Download and upload files - Enables users to download and upload files from websites.
    • Provide Security Features - Warns users about unsafe websites and helps protect personal information.
    • Store browsing data - Saves bookmarks, browsing history, passwords, and cookies for easier access.

  6. What is DNS and why it is important?

    7. What is DNS?

    It's a lot like a phonebook for the internet. It is a large network of servers located across the globe that contain a distributed database of domain names and IP addresses. It's responsible for matching the right domain name with its corresponding IP address.
