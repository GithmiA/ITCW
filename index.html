  5. Role of Web Browsers.

    6. A web browser is software that allows users to access, view, and interact with websites on the internet safely and efficiently.

    • Access websites -Opens websites using their web addresses (URL).
    • View web pages - Shows text,images ,videos and other content.
    • Interact with web content - Allows users to interact with web elements like links, forms, and buttons.
    • Run web applications - Supports interactive websites such as email, online banking, and social media.
    • Download and upload files - Enables users to download and upload files from websites.
    • Provide Security Features - Warns users about unsafe websites and helps protect personal information.
    • Store browsing data - Saves bookmarks, browsing history, passwords, and cookies for easier access.

  6. What is DNS and why it is important?

    7. What is DNS?


    DNS - Domain Name System

    It's a lot like a phonebook for the internet. It is a large network of located across the
    globe that contain a distributed database of domain names and IP addresses.
    It's responsible for matching the right domain name with its corresponding IP address.

    • IP Address (Internet Protocol)

      Ex: 192.168.1.1
      An IP address is a unique numerical identifier assigned to every device connected to the Internet or a local network. It is a set of unique numbers separated by periods.

    • Domain Name

      Ex: www.google.com
      Domain name is used to uniquely identify a web site.An IP address is used to uniquely identify a computer on the Internet. What the Domain Name Server does is converting the domain name to an IP address.

    Why is DNS Important?

    DNS is the backbone of the Internet. Without it, the Internet as we know it would collapse.
    Since DNS can resolve a domain name to the correct IP address, it enables quick access to the website we are looking for.

HTML Tags

Tag Name Purpose and Example
01. <head> A container for metadata and machine-readable information that helps browsers and search engines process the web page correctly.
The following elements can go inside the <head> element:
  • <title>
    Defines the document's title that is shown in a browser's title bar or a page's tab.

    • <head>

    <title>My Web Page</title>

    </head>

  • <link>

    • <head>

    <link rel="stylesheet" type="text/css" href="styles.css">

    </head>

  • <style>
    Contains style information for a document or part of a document.

    • <head>

    <style>

    h1{ color:red; }

    </style>

    </head>
02. <body> The <body> element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

<body>

<h1>This is a heading</h1>

<p>This is a paragraph</p>

<img src="image.jpg" alt="An image" width="24" height="39">

<a href="https://www.example.com">This is a link</a>

</body>
03. <h1> to <h6> Represent six levels of section headings. <h1> is the highest section level and <h6> is the lowest.

<body>

<h1>This is a heading</h1>

<h2>This is a heading</h2>

<h3>This is a heading</h3>

<h4>This is a heading</h4>

<h5>This is a heading</h5>

<h6>This is a heading</h6>

</body>
04. <hr> Represents a horizontal rule in an HTML document.

<body>

<hr style="border: 1px solid blue; margin-top: 20px; margin-bottom: 20px; width: 80%;">

</body>
05. <br> Represents a line break in an HTML document.

<body>

<p>

This is a paragraph.<br/>

This is another paragraph.

</p>

</body>
06. <button> The <button> tag defines a clickable button.

<body>

</body>
