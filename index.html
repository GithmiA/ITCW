    1. The user opens the web broweser and enters the URL.

    2. The web browser sends the URL to the DNS server.

    3. The DNS server translates the name in the URL into the corresponding IP address.

    4. The request is sent to the web server with that IP address.

    5. The web server returns an HTML data stream.

    6. The web browser displays the HTML web page.
